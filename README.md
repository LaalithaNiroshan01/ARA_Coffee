## ✨ Overview

ARA Coffee is a high-end, immersive landing page designed for a premium coffee brand. The project showcases editorial-style aesthetics with cinematic scroll animations, smooth interactions, and a luxurious visual identity — built entirely with cutting-edge front-end technologies.

## 🚀 Features

- 🎬 **GSAP Scroll Animations** — Cinematic, scroll-triggered reveal animations throughout every section
- 🌊 **Lenis Smooth Scroll** — Butter-smooth scrolling experience for an immersive feel
- 🖱️ **Custom Cursor** — Interactive branded cursor that reacts to hover states
- 📜 **Infinite Marquee** — Animated text marquee for brand messaging
- 🎨 **Editorial Design** — Film-textured, asymmetrical layouts with premium typography
- 📱 **Responsive Layout** — Optimized for all screen sizes
- ⚡ **Vite-Powered** — Lightning-fast development server and build pipeline

## 🏁 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18 or higher
- npm v9 or higher

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/LaalithaNiroshan01/ARA_Coffee.git
   cd ARA_Coffee
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:5174
   ```

### Build for Production

```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```

- **Animations** — GSAP `ScrollTrigger` for scroll-based reveals and parallax effects
